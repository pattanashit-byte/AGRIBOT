# 🌾 AGRIBOT - AI Agricultural Advisory System

## 📌 Overview

AGRIBOT is an AI-powered agricultural advisory system designed to help farmers with:

* Crop recommendations 🌱
* Disease detection 🦠
* Smart chatbot assistance 🤖

## 🚀 Features

* 🌿 Crop Recommendation System
* 🧠 Machine Learning-based Disease Detection
* 💬 AI Chatbot for farmer queries
* 📊 User-friendly dashboard

## 🛠️ Tech Stack

* Python
* Flask
* Machine Learning (Scikit-learn / TensorFlow)
* HTML, CSS, JavaScript

## 📂 Project Structure

```
AGRIBOT/
│── app.py
│── requirements.txt
│
├── routes/
├── src/
├── static/
├── templates/
├── utils/
```

## ▶️ How to Run

```bash
git clone https://github.com/your-username/AGRIBOT.git
cd AGRIBOT
pip install -r requirements.txt
python app.py
```
